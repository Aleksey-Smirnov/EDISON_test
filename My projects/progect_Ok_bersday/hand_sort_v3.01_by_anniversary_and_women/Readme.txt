    Программа find_friends.py для поиска друзей именинниц и юбиляров из соцсети "Одноклассники". В папке id_html/"город" - список именников в html формате. В папке date/"город" - списки друзей для загрузки на Таргет.

    17.04.2021 Добавлена возможность передачи параметров с командной строки в формате "python find_friends.py -c [список городов через пробел(belokuriha ziryanovsk)] -d [список дат через пробел(19_04 20_04)]
    
     28.04.2021 def_get_selenium.py - модуль эмулятор броузера, загрузка списка именинников с ОК
