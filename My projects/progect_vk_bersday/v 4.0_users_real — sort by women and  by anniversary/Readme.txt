Программа bd_friends_user_real - нахождение id друзей именинниц и id друзей юбиляров в социальной сети "Вконтакте"

bd_data.txt - файл с данными. В формате city - индификаторы городов, birth_day - дата в числовом выражении, birth_month - месяц в числовом выражении, access_token -актуальный токен.

В папке data выводятся результаты поиска  в формате город_назначение_дата_месяц_txt файлов. 
-users_id_opens -id именинников с открытыми профилями (актуальные + неактуальные), 
-users_id_real - id именников с актуальными профилями (закрытые + открытые), 
-friends_id_opens - id друзей именинников, выборка по актуальным профилям.
